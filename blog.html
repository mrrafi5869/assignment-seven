1. What's the difference of local storage and session storage

Local storage Session storage
Local storage stores data with no expiration date. Session storage is destroyed once the user closes the browser

1. What's the difference of Global scope and block scope

Global Scope Block Scope
When a variable is globally scoped, it means it is available from anywhere in your program A block scoped variable means that the variable defined within a block will not be accessible from outside the block.
